# Jogo da Velha (Java)
Jogo da Velha feito na linguagem Java utilizando [IntelliJ IDEA](https://www.jetbrains.com/idea/), para ser jogado via terminal / prompt de comando.

### Compilar:
`$ javac src/com/niemietz/JogoDaVelha.java` 

### Executar:
`$ java JogoDaVelha`
