# Jogo da Velha (Java)
Jogo da Velha feito na linguagem java, para ser jogado via terminal / prompt de comando.

### Compilar:
`$ javac src/com/niemietz/JogoDaVelha.java` 

### Executar:
`$ java JogoDaVelha`
